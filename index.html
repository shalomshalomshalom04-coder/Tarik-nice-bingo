Quick start

  1. If your game is a static site, upload index.html, css/, js/, and assets/ to the repository root.
  2. Enable GitHub Pages: Repository Settings → Pages → Source: main branch / / (root). Wait a few minutes for the site to publish.
  3. Your site will be available at: https://shalomshalomshalom04-coder.github.io/Tarik-nice-bingo/

Want a free custom domain?

You can use a free domain from Freenom (example.tk/.ml/.ga/.cf/.gq) and point it to GitHub Pages. Add a CNAME file with your domain (e.g. www.example.tk) and configure the domain's DNS to point to GitHub Pages IP addresses. See the README in this repo for step-by-step instructions.
